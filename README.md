# ikn_tfidf_knn

Presiden Bapak Ir. H. Joko Widodo merencanakan pemindahan dan pembangunan Ibu Kota Negara Nusantara secara bertahap dari tahun 2024 hingga 2045. Karena itu, ini menjadi topik diskusi dan mengundang banyak tanggapan, terutama dari masyarakat Indonesia. Sangat menarik untuk diperdebatkan di media sosial, termasuk Tiktok, tentang masalah pemindahan ibu kota negara. Pada dasarnya, media sosial digunakan untuk berekspresi. Tidak diragukan lagi, penggunaan media sosial ini berfungsi sebagai layanan dan fasilitas yang dapat digunakan untuk menyelesaikan masalah politik atau masalah yang sedang dibicarakan. Oleh karena itu, tujuan penelitian ini adalah untuk memeriksa perasaan masyarakat Indonesia tentang pemindahan Ibu Kota Nusantara. Penelitian ini menggunakan Algoritma K-Nearest Neighbor (KNN) dengan pembobotan TF-IDF. Hasil penelitian menunjukkan bahwa kinerja analisis dari kedua metode tersebut dengan nilai precision untuk kelas negatif 97%, sedangkan precision untuk kelas positif adalah 67%. Model memiliki performa yang baik dengan akurasi yang cukup tinggi sebesasr 78%.
